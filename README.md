# Игра "Лабиринт"

Простая игра-лабиринт с тремя уровнями и скримером в конце.

## Как играть

1. Откройте файл `index.html` в вашем браузере
2. Управляйте персонажем с помощью клавиш стрелок или WASD
3. Доберитесь до красной точки (финиш) на каждом уровне
4. После прохождения всех трех уровней вас ждет сюрприз!

## Установка

1. Скачайте или клонируйте репозиторий
2. Добавьте файл изображения для скримера с именем `screamer.jpg` в корневую папку 
3. (Опционально) Добавьте звук скримера с именем `scream.mp3`
4. Откройте `index.html` в браузере

## Требования

- Современный веб-браузер с поддержкой HTML5 и JavaScript
- Устройство с клавиатурой для управления
